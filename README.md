# 7克隆 （7Gost）
* 7ghost是一款基于PHP的网站反向代理程序。能够快速高效的反向代理所指定的网站。
* 并拥有丰富的内容替换、请求头设置。让没有主机的朋友也可以反向代理和加速你的网站。

## 功能
* 指定网站完全代理
* 自定义页面代理
* 域名替换
* 内容替换
* 静态缓存
* cookies自定义
* 浏览器标识自定义
* referer自定义

## 运行环境
 * 推荐在Linux服务器上运行
 * 推荐使用Nginx或Apache运行
 * 最小运行内存不因低于64 MB
 * 最低PHP版本不应低于PHP5.4

## 说明
* 后台地址为：<br>\_admin\</br>
* 默认登录密码：<br>123456</br>
* 可编辑: <br>\_admin\data\config.php</br> 修改。
